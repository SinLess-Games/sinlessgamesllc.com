# Helix AI Discord Bot Service

---

This is a Discord bot service that is used to provide a variety of services to the SinLess Games 
LLC Official discord Server.

## Specification

---

Helix's discord bot (discord.js) runs side by side with the Express.js backend, The dashboard runs decoupled
using api routes to fetch data. TLS is used to secure traffic between the two. The dashboard runs on React.js 
to provide the most intuitive and responsive experience for our users.

## Getting Started

---


### Prerequisites

You will need to have the following installed on your machine:
- [Node.js](https://nodejs.org/en/) (v16 or newer)
- [Git](https://git-scm.com/downloads)

### setup

1. Clone the repository
2. Run `npm install` to install the dependencies

### Running the bot

1. Run `npm run start` to start the bot
2. Run `npm run dev` to start the bot in development mode
3. Run `npm run build` to build the bot
4. Run `npm run lint` to lint the bot
5. Run `npm run test` to test the bot
6. Run `npm run docs` to generate the documentation
